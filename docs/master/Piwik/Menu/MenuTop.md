### `addEntry()` <a name="addEntry"></a>

Adds a new entry to the TopMenu.

#### Signature

- It is a **public static** method.
- It accepts the following parameter(s):
    - `$topMenuName`
    - `$data`
    - `$displayedForCurrentUser`
    - `$order`
    - `$isHTML`
    - `$tooltip`
- It does not return anything.
