Piwik Plugins API documentation generator

## Install

 * Clone repository
 * git submodule init
 * git submodule update
 * php composer.phar install

## Usage

 * Execute `./generate.sh`
 * Commit
 * Push

or just execute `./generateAndPush.sh`
